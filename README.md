# StudentOrganizer

**StudentOrganizer** — это приложение на базе Node.js и Electron для управления расписанием занятий, напоминаниями и успеваемостью студентов.

## Основные функции

### Расписание занятий
- Ведение расписания для двух недель (первая и вторая недели).
- Указание порядка занятий в течение дня.
- Возможность добавлять, редактировать и удалять занятия.

### Напоминания
- Добавление напоминаний с описание и датой дедлайна.
- Напоминания отображаются в табличном виде.
- Сортировка напоминаний по дате (ближайшие сверху).

### Успеваемость
- Учет успеваемости по предметам из расписания.
- Автоматическое объединение одинаковых предметов.

## Установка

1. Клонируйте репозиторий:
    ```cmd
    git clone https://github.com/your-repo/student-organizer.git
    ```

2. Перейдите в папку проекта:
    ```cmd
    cd student-organizer
    ```

3. Установите зависимости:
    ```cmd
    npm install
    ```

4. Запустите приложение:
    ```cmd
    npm start
    ```

## Структура проекта

- `index.html`: Главный HTML-файл с пользовательским интерфейсом.
- `main.js`: Основной скрипт, управляющий логикой приложения.
- `app.js`: Основной процесс Electron.

## Использование

### Добавление занятия
1. Нажмите на кнопку "Добавить занятие".
2. Заполните поля: название, порядок, день недели, номер недели и примечание.

### Удаление занятия
1. Нажмите на иконку корзины рядом с занятием.

### Добавление напоминания
1. Нажмите на кнопку "Добавить напоминание".
2. Заполните поля: Описание и дата дедлайна.

### Удаление напоминания
1. Нажмите на иконку корзины рядом с напоминанием.

## Требования

- Node.js версии 14 и выше.
- Electron версии 21 и выше.
